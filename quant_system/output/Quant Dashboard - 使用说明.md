# Quant Dashboard v2 · 使用说明

## 打开方式

浏览器直接打开 `Quant Dashboard v2.html`，无需服务器、无需安装任何依赖。

```
双击 Quant Dashboard v2.html → 浏览器自动打开
```

推荐 Chrome / Edge / Safari 最新版。Firefox 可用但 `backdrop-filter` 毛玻璃效果略弱。

## 各模块说明

### 1. 顶栏

| 元素 | 说明 |
|------|------|
| `QUANT_SYS v0.2` | 系统名称和版本 |
| `● LIVE` | 绿色脉冲点 = 系统运行中 |
| `股票池: HS300` | 当前选股范围（沪深300） |
| `策略: 多因子` | 当前使用的策略模板 |
| `调仓: 每周` | 调仓频率 |
| 右侧时间 | 当前日期和时钟 |

### 2. 行情滚动条

- **自动滚动**：12 只持仓股的实时价格横向轮播
- **悬停暂停**：鼠标移到滚动条上自动暂停，方便查看
- **颜色含义**：
  - 亮黄色加粗 = 股票名称
  - 白色 = 最新价
  - 绿色 = 上涨，红色 = 下跌
- **数据刷新**：每 2 秒价格微调（演示模式）

### 3. KPI 指标卡（6 个）

| 指标 | 含义 | 颜色 |
|------|------|------|
| 总权益 | 当前账户总资产 | 绿色 ↑ |
| 当日盈亏 | 今日浮动盈亏 | 绿涨红跌 |
| 夏普比率 | 风险调整后收益（>1 良好，>2 优秀）| 绿色 |
| 最大回撤 | 从历史最高点算起的最大跌幅 | 琥珀色 ⚠️ |
| VaR(95% 1日) | 95% 置信度单日最大可能亏损 | 琥珀色 ⚠️ |
| 胜率 | 盈利交易占比 | 绿色 |

### 4. 蜡烛图（主图区）

- **K 线**：绿色空心 = 阳线（涨），红色实心 = 阴线（跌）
- **影线**：细线上下延伸 = 当日最高/最低价
- **成交量柱**：图底半透明柱，对应每根 K 线的成交量
- **均线**：
  - 琥珀色 ── = 5 周期均线
  - 紫色 ── = 10 周期均线
- **时间切换**：点击 `1周` `1月` `3月` `1年` 缩放图表
- **十字光标**：鼠标悬停 K 线上 → 显示 O/H/L/C 数据浮层

### 5. 持仓表

| 列 | 含义 |
|----|------|
| 股票 | 代码 + 名称 |
| 权重 | 该股占总资金比例 |
| 盈亏 | 持仓至今浮动盈亏 |
| 信号 | 买入 / 持有 / 卖出 |
| 持日 | 已持有天数 |

- 鼠标悬停行 → 背景轻微高亮
- 信号颜色：绿色=买入，红色=卖出，灰色=持有

### 6. 风控面板（4 个）

每项包含数值 + 进度条：

| 指标 | 进度条颜色逻辑 |
|------|--------------|
| VaR 95% | 琥珀色（接近上限时变红）|
| 最大回撤 | 琥珀色（回撤越大越接近红色）|
| 杠杆率 | 绿色（当前为 0 = 无杠杆）|
| 相关性 | 绿色（正常范围）|

### 7. 底栏

- **数据源**：当前使用 baostock，回退链 akshare → efinance
- **最后更新**：数据刷新时间
- **下次调仓**：下个调仓日
- **推送/测试**：PushPlus 就绪状态 / 测试通过数

## 模拟数据说明

当前版本使用**前端模拟数据**展示，实际接入后端后替换以下数据源：

| 数据 | 当前 | 接入后端后 |
|------|------|-----------|
| KPI 数值 | `useState` 三角函数模拟波动 | `quant_system` API 返回真实账户数据 |
| 蜡烛图 | `generateCandles()` 随机生成 | baostock 拉取真实 K 线 |
| 行情滚动 | `setInterval` 随机微调 | WebSocket 推送实时行情 |
| 持仓表 | 硬编码 10 只示例 | 后端返回实际持仓 |
| 风控指标 | 固定值 | 实时计算 VaR/回撤/相关性 |

## 技术栈

- **React 18.3.1** — UI 框架（CDN 加载，无需构建）
- **Babel Standalone 7.29.0** — 浏览器内编译 JSX
- **Canvas API** — 蜡烛图绘制
- **CSS Grid + backdrop-filter** — 响应式玻璃拟态布局
- **无外部依赖** — 单 HTML 文件，双击即开

## 常见问题

**Q: 为什么蜡烛图数据不会变？**
A: 当前为演示模式，数据在页面加载时一次生成。接入后端 WebSocket 后实现实时更新。

**Q: 行情滚动条可以调速度吗？**
A: 打开文件搜索 `tickerScroll 40s`，修改 `40s` 数值（越小越快）。

**Q: 如何切换策略模板？**
A: 顶栏「策略: 多因子」显示当前策略。后端模式下通过 CLI 切换：
```bash
python -m quant_system.main architect --template momentum
```

**Q: 页面打开白屏？**
A: 可能是 CDN 加载超时（unpkg.com 国内访问不稳）。等待 10 秒或刷新重试。
