## 一、缠论核心概念

缠论（缠中说禅理论）是中国最具影响力的本土技术分析体系之一，其核心思想是**走势终完美**——任何级别的走势最终都会完成，而完成之前必然经历趋势背驰。

缠论从K线出发，自底向上构建7个层级：

```
K线包含处理 → 分型 → 笔 → 线段 → 中枢 → 背驰 → 买卖点
   (L1)       (L2)   (L3)  (L4)   (L5)   (L6)    (L7)
```

### 1.1 K线包含处理 (Level 1)

当一根K线完全被相邻K线的高低价范围覆盖时，需要合并处理：

| 趋势方向 | 合并规则 | 说明 |
|----------|---------|------|
| 上升趋势 | 取**高高** | high取max, low取max |
| 下降趋势 | 取**低低** | high取min, low取min |

合并后的K线消除了包含关系，为后续分型识别做准备。

### 1.2 分型 (Level 2)

分型是最基本的形态单元：

```
顶分型:         底分型:
    /\              \/
   /  \              \  /
  /    \              \/
```

| 类型 | 定义 | 市场含义 |
|------|------|---------|
| **顶分型** | 中间K线的高点最高，低点也最高 | 多空转换，顶部信号 |
| **底分型** | 中间K线的低点最低，高点也最低 | 空多转换，底部信号 |

分型必须由**经过包含处理后的三根连续K线**构成。

### 1.3 笔 (Level 3)

笔是缠论最基本的走势单元：

```
向上笔:  底分型 → 顶分型  (从低到高的运动)
向下笔:  顶分型 → 底分型  (从高到低的运动)
```

**笔的构建规则**:
1. 笔必须由相邻的**异类分型**连接
2. 两个分型之间至少包含 **5 根独立K线**（不含包含关系）
3. 向上笔的顶必须**高于**底分型
4. 向下笔的底必须**低于**顶分型
5. 同方向的两个笔之间至少有一个相反方向的笔

### 1.4 线段 (Level 4)

线段由**至少3笔**构成，是更高级别的走势单元：

```
向上线段:  底分型 → (向上笔) → (向下笔) → (向上笔) → 顶分型
向下线段:  顶分型 → (向下笔) → (向上笔) → (向下笔) → 底分型
```

线段内部包含奇数笔，方向由第一笔决定。

### 1.5 中枢 (Level 5)

**中枢是缠论最核心的概念**——由至少3段连续线段的重叠区间构成：

```
中枢上沿 (ZG) = min(线段1高点, 线段2高点, 线段3高点)
中枢下沿 (ZD) = max(线段1低点, 线段2低点, 线段3低点)

要求: ZD < ZG   (有重叠才是中枢)
```

中枢的意义：
- **价格引力中心**: 价格围绕中枢震荡
- **支撑与阻力**: 中枢上下沿构成天然的支撑/阻力位
- **趋势定义**: 中枢上移 = 上升趋势，中枢下移 = 下降趋势
- **级别递归**: 小级别中枢可以扩展为大级别中枢

### 1.6 背驰 (Level 6)

背驰是趋势衰竭的信号，也是缠论买卖点的核心判断依据：

| 类型 | 价格 | MACD | 含义 |
|------|------|------|------|
| **顶背驰** | 价格创新高 | MACD红柱面积减小, DIFF不再新高 | 上涨力度衰竭，即将转折向下 |
| **底背驰** | 价格创新低 | MACD绿柱面积减小, DIFF不再新低 | 下跌力度衰竭，即将转折向上 |

我们使用 **MACD面积比较法** 和 **DIFF值比较法** 双重确认背驰信号。

### 1.7 三类买卖点 (Level 7)

缠论定义了三种经典的买入点和三种卖点：

#### 买入点

| 类型 | 位置 | 条件 | 可靠性 |
|------|------|------|--------|
| **一买 (B1)** | 下跌趋势末端 | 底背驰 + 跌破最后一个中枢 | ⭐⭐⭐ 高赔率，需确认 |
| **二买 (B2)** | 一买后回踩 | 回踩不破一买低点 + 底分型 | ⭐⭐⭐⭐⭐ 最可靠 |
| **三买 (B3)** | 中枢上方 | 突破中枢上沿后回踩不入中枢 | ⭐⭐⭐⭐ 趋势确认后介入 |

#### 卖出点

| 类型 | 位置 | 条件 | 可靠性 |
|------|------|------|--------|
| **一卖 (S1)** | 上涨趋势末端 | 顶背驰 + 升破最后一个中枢 | ⭐⭐⭐ 高赔率，需确认 |
| **二卖 (S2)** | 一卖后反弹 | 反弹不破一卖高点 + 顶分型 | ⭐⭐⭐⭐⭐ 最可靠 |
| **三卖 (S3)** | 中枢下方 | 跌破中枢下沿后反抽不入中枢 | ⭐⭐⭐⭐ 趋势确认后退场 |

```
    买入点示意:                    卖出点示意:
    
    B3  ──╮  不进入中枢            S3  ──╮  不进入中枢
          │                              │
        中枢上沿                        中枢下沿
    ════ ════ ════                ════ ════ ════
        中枢下沿                        中枢上沿
          │                              │
    B2 ──╯  回踩不破               S2 ──╯  反弹不破
          │                              │
    B1 ──╯  背驰转折               S1 ──╯  背驰转折
```

### 2.2 信号置信度

每个买卖信号都带有**置信度评分** (0.0 - 1.0)：

| 置信度 | 含义 | 建议操作 |
|--------|------|---------|
| > 0.7 | 强信号，多重确认 | 按信号满仓/空仓 |
| 0.5 - 0.7 | 中等信号 | 按信号操作但降低仓位 |
| 0.3 - 0.5 | 弱信号 | 结合其他指标确认 |
| < 0.3 | 噪声 | 忽略 |


```
  ↓

  ↓
```

## 五、参数调优指南

### 5.1 关键参数

| 参数 | 默认值 | 说明 | 调优建议 |
|------|--------|------|---------|
| `MIN_BI_KLINE_COUNT` | 5 | 笔的最小K线数 | 降低→更多笔更敏感；提高→更少笔更稳健 |
| `MIN_HUB_OVERLAP` | 3 | 中枢最小段数 | 3为标准中枢，5为大级别中枢 |
| `MACD_FAST` | 12 | MACD快线 | 震荡市降低(8-10)，趋势市不变 |
| `MACD_SLOW` | 26 | MACD慢线 | 同上 |
| `MACD_SIGNAL` | 9 | MACD信号线 | 同上 |
| `DEFAULT_LOOKBACK` | 500 | 回溯K线数 | 根据分析级别调整 |

### 5.2 不同市场环境的参数设置

| 市场状态 | MACD参数 | 笔K线数 | 置信度阈值 | 特征 |
|----------|---------|---------|-----------|------|
| **趋势市 (牛市)** | (12, 26, 9) 标准 | 5 | 0.5 | 中枢不断上移，三买机会多 |
| **趋势市 (熊市)** | (10, 22, 8) 敏感 | 5 | 0.4 | 中枢不断下移，三卖机会多 |
| **震荡市** | (14, 28, 10) 钝化 | 7 | 0.6 | 中枢延伸为主，减少假信号 |
| **高波动市** | (16, 30, 10) 稳健 | 7 | 0.65 | 避免被剧烈波动洗出 |

### 5.3 多级别联立

缠论强调**多级别联立**，不同级别信号互相验证：

| 级别 | K线周期 | 适用场景 |
|------|--------|---------|
| 日线 | 日K | 中期趋势判断，买卖点确认 |
| 60分钟 | 60分K | 短期波段操作，精确入场 |
| 30分钟 | 30分K | 日内短线，T+0操作 |
| 周线 | 周K | 大趋势判断，仓位管理 |

**联立原则**: 大级别定方向，小级别找买卖点。周线向上时只做多（买点有效），周线向下时只做空（卖点有效）。

## 六、风险提示与策略局限

### 6.1 风险提示

1. **假信号风险**: 震荡市中中枢反复延伸，容易产生假背驰信号
2. **滞后性**: 笔和线段需要K线数量确认，买卖点可能滞后于实际转折
3. **级别混淆**: 不同级别的中枢嵌套会干扰判断，需要严格区分级别
4. **单边行情**: 极端单边行情中中枢可能无法形成，导致买卖点缺失
5. **数据质量**: K线数据的前复权/后复权会影响分型和笔的计算

### 6.2 策略优化方向

| 方向 | 说明 | 优先级 |
|------|------|--------|
| 成交量确认 | 买点伴随放量，卖点伴随缩量 | 🔴 高 |
| 多级别联立 | 日线+60分钟+30分钟三级别共振 | 🔴 高 |
| 趋势线辅助 | 结合通道线和趋势线过滤假突破 | 🟡 中 |
| 波动率自适应 | 根据ATR动态调整中枢高度阈值 | 🟡 中 |
| 机器学习增强 | 用历史信号训练分类器提高置信度 | 🟢 低 |
| 实时推送 | 买卖点触发时通过钉钉/微信推送 | 🟢 低 |

### 6.3 与ETF融合策略的配合

本缠论策略可与项目中的ETF融合轮动策略 (`etf_fusion_signals.py`) 互补使用：

| 维度 | 缠论策略 | ETF融合策略 |
|------|---------|------------|
| 选股 | 技术形态选时 | 动量因子选股 |
| 信号 | 买卖点触发 | 月度轮动调仓 |
| 适合市场 | 震荡+趋势 | 趋势+动量 |
| 调仓频率 | 按信号触发 | 月度固定 |
| 互补方式 | 缠论信号确认融合策略的调入/调出时机 | 融合策略提供候选池，缠论确定入场点 |
