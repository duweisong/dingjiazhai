# AO + Keltner Channel 策略详解

## 一、策略起源

该策略源自微信公众号「数据科学实战」的文章《亏损 40% 的"烂股"，Python 策略却反赚 164%？》。

原文用 Python 回测了 AES（纽交所全球电力公司），该股 25 年买入持有亏损 39.8%，但一套 AO + KC 策略却实现 164% 收益。核心发现：**胜率仅 33%，靠"大赚小亏"的盈亏不对称取胜。**

本项目将该策略移植到 A 股市场，在通达信本地数据上完成沪深 300 全量验证。

## 二、指标说明

### AO（Awesome Oscillator，动量震荡指标）

由 Bill Williams 提出，衡量市场动能：

```
中位价 = (最高价 + 最低价) / 2
AO = SMA(中位价, 5) - SMA(中位价, 34)
```

**入场判断**：当前 AO > 5 根K线前的 AO → 短期动量在回升。

> 注意：不要求 AO > 0，只要求 AO 比 5 天前更高。这能捕捉"负值收窄→即将转正"的拐点。

### Keltner Channel（肯特纳通道）

基于波动率的通道指标：

```
中轨 = EMA(典型价, 20)
ATR = 平均真实波幅(20)
上轨 = 中轨 + 2 × ATR
下轨 = 中轨 - 2 × ATR
```

**离场判断**：收盘价跌破下轨 → 价格被打到波动率极端位置，止损离场。

> 与布林带的区别：KC 用 ATR 而非标准差，对极端跳空更敏感。

## 三、交易流程

```
                    开始
                     │
              ┌──────▼──────┐
              │  AO 在上升？  │──No──→ 等待
              └──────┬──────┘
                     │Yes
              ┌──────▼──────┐
              │  冷却期已过？ │──No──→ 等待
              └──────┬──────┘
                     │Yes
              ┌──────▼──────┐
              │   计算仓位    │
              │ 买入（次日开盘）│
              └──────┬──────┘
                     │
              ┌──────▼──────┐
              │  每日检查：   │
              │  1.跌破KC下轨?│──Yes──→ 卖出
              │  2.跌破固定止损│──Yes──→ 卖出
              │  3.跌破移动止损│──Yes──→ 卖出
              │  4.AO转负+破轨│──Yes──→ 卖出
              │  以上皆否     │──No──→ 继续持有
              └─────────────┘
```

## 四、仓位计算

```
风险金额 = 账户资金 × 2%
止损距离 = 2.5 × ATR（当前波动率）
理论股数 = 风险金额 ÷ 止损距离
仓位上限 = 账户30% ÷ 股价
实际买入 = min(理论股数, 仓位上限)
```

**例如**：资金 10 万，股价 $50，ATR=$2
- 风险金额 = $2,000
- 止损距离 = $5（2.5×$2，即跌 10% 止损）
- 理论买入 = 400 股
- 仓位上限 = 600 股（$30,000÷$50）
- 实际买入 = 400 股（$20,000 仓位）

## 五、适用场景

| ✅ 适合 | ❌ 不适合 |
|---------|----------|
| 高位回落后震荡筑底 | 单边持续阴跌无反弹 |
| 有明显箱体波动区间 | 日均波动 < 1% 的横盘 |
| 间歇性 20%+ 波段反弹 | 持续缩量创新低 |
| 新能源、医药、消费电子等波动板块 | 部分大金融、公用事业等低波动板块 |

## 六、风险提示

- 胜率 35-50%，连续亏损 5-8 笔是正常现象
- 最大回撤可达账户的 10-20%
- 单边牛市中必然跑输买入持有
- 历史回测不代表未来表现
- 实盘前请充分测试和验证
