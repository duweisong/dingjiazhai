# AO + Keltner Channel 均值回归交易策略

> 低胜率 × 高盈亏比 = 稳定盈利 | 沪深300 实测 95.3% 股票盈利

基于 **Awesome Oscillator（动量）** 和 **Keltner Channel（波动率）** 的量化交易系统，回测 2021-2026 年沪深 300 全部成分股，**298 只中 284 只策略正收益（95.3%）**，平均超额收益 +26.3%。

---

## 策略逻辑

```
买入：AO 动量回升（当前值 > 5根K线前的值）
卖出：收盘跌破 KC 下轨 | 固定止损 | 移动止损 | AO转负
```

| 要素 | 规则 |
|------|------|
| 🟢 **入场** | AO 拐头向上 + 冷却期 ≥ 3 根K线 |
| 🔴 **离场** | 收盘 < KC下轨 / 固定止损(2.5×ATR) / 移动止损(3×ATR) |
| 📦 **仓位** | 动态ATR仓位 = min(风险金÷止损距离, 资金30%÷股价) |
| 🛑 **风控** | 单笔最大亏损 2% | 单票仓位上限 30% |

> 核心哲学：**截断亏损，让利润奔跑。** 不靠高胜率，靠盈亏比取胜。

## 参数调优

编辑 `ao_kc_strategy.py` 中的 `Cfg` 类：

```python
class Cfg:
    AO_SHORT = 5        # AO 短周期
    AO_LONG = 34        # AO 长周期
    AO_SHIFT = 5        # 动量比较偏移
    KC_PERIOD = 20      # KC 周期
    KC_MULT = 2.0       # KC 宽度倍数
    MIN_BARS = 3        # 最小交易间隔
    RISK_PCT = 0.02     # 单笔风险 2%
    STOP_ATR = 2.5      # 止损 ATR 倍数
    TRAIL_ATR = 3.0     # 移动止损 ATR 倍数
    MAX_POS = 0.30      # 最大仓位 30%
```

## 注意事项

- 本策略为**均值回归**策略，在**震荡市/下跌反弹市**表现最佳
- 在单边大牛市中可能跑输买入持有
- 胜率约 35-50%，需接受连续亏损的心理考验
- 所有回测基于历史数据，不构成投资建议
- 实盘前请自行充分研究和测试
