# 仓位模型设计文档

## 1. 信号提取

从 9 个季度国家队持仓数据中提取 3 个维度：

### 1.1 当前信心指数 (weight: 50%)

```
conviction = (增持数 - 减持数) / (增持数 + 减持数)
```

- 范围：-1 到 +1
- 当前值（2026Q1）：-0.112

### 1.2 趋势方向 (weight: 30%)

```
trend = conviction(当前) - conviction(上季度)
```

- 正值 = 国家队在加仓，趋势向好
- 负值 = 国家队在减仓，趋势变差
- 归一化到 -1 ~ +1

### 1.3 加速度 (weight: 20%)

```
acceleration = trend(当前) - trend(上季度)  
             = conviction(Q) - 2*conviction(Q-1) + conviction(Q-2)
```

- 正值 = 减仓速度在放缓或加仓在加速
- 负值 = 减仓在加速

## 2. 仓位公式

```python
raw_score = 0.50 * conviction_norm 
          + 0.30 * trend_norm 
          + 0.20 * acceleration_norm

# Sigmoid 平滑到 [0, 1]
position = 1.0 / (1.0 + exp(-k * raw_score))

# k=3 使输出在 0.05~0.95 范围内敏感
```

### 仓位建议区间

| raw_score 范围 | 仓位 | 说明 |
|---------------|------|------|
| > +1.0 | 90-100% | 全力做多 |
| +0.3 ~ +1.0 | 70-90% | 积极 |
| -0.3 ~ +0.3 | 40-60% | 中性 |
| -1.0 ~ -0.3 | 20-40% | 保守 |
| < -1.0 | 0-20% | 大幅减仓 |

## 3. 当前数据

基于 2026-06-12 的 9 季度数据：

```
2023Q2: +0.380 (BULL)
2023Q4: +0.279 (BULL)
2024Q2: +0.206 (BULL)
2024Q3: +0.279 (BULL)
2024Q4: -0.054 (NEUTRAL) ← 拐点
2025Q2: +0.042 (NEUTRAL)
2025Q3: +0.000 (NEUTRAL)
2025Q4: +0.040 (NEUTRAL)
2026Q1: -0.112 (BEAR)    ← 当前
```

```python
conviction = -0.112
trend = -0.112 - 0.040 = -0.152
acceleration = -0.152 - (0.040 - 0.000) = -0.192

raw_score = 0.50 * (-0.112) + 0.30 * (-0.152) + 0.20 * (-0.192)
          = -0.056 + -0.046 + -0.038
          = -0.140

position = sigmoid(3 * -0.140) = 0.396 → 40%
```

**当前建议仓位：40%**

## 4. 平滑机制

- 季度间仓位变化不超过 30%（避免剧烈摇摆）
- 连续两季同方向 → 仓位按 1.2x 加速调整
- 方向反转 → 仓位按 0.5x 减速调整（谨慎确认）
