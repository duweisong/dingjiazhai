# nt-position-sizer · 国家队仓位判定器

基于国家队（社保/养老金/汇金/证金）季度持仓变化，输出 **0-100% 连续仓位建议**。

## 核心逻辑

```
9 季度持仓数据 → 多维度信号提取 → 仓位公式 → 0-100% 仓位比例
```

### 信号维度

| 维度 | 来源 | 权重 |
|------|------|------|
| 当前信心指数 | 最新季度买卖比 | 50% |
| 趋势方向 | 连续 2 季度变化方向 | 30% |
| 加速度 | 变化速度（二阶导） | 20% |

### 仓位映射

- 信心 > +0.20 + 趋势向上 → **80-100%**（积极持仓）
- 信心 0~0.20 → **60-80%**（正常持仓）
- 信心 -0.10~0 → **40-60%**（中性偏保守）
- 信心 -0.20~-0.10 + 趋势向下 → **20-40%**（减仓）
- 信心 < -0.20 → **0-20%**（大幅减仓/空仓）

## 数据源

复用父项目 `c:/AI/` 下的：
- `national_team_tracker.py` — 数据拉取
- `.cache/national_team/` — 9 季度持仓 parquet

## 快速开始

```bash

python src/position_engine.py

# 回测仓位策略 vs 基准
python src/position_engine.py --backtest

# 月度自动更新（Windows 任务计划）
python src/position_engine.py --live
```
